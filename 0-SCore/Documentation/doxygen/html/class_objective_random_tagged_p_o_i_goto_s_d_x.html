Detailed Description

This class is meant to be a replacement for ObjectiveRandomPOIGoto, with these differences:

  • You can specify which tags a POI must have, to be a target for the quest
  • You can specify which tags a POI must NOT have, to be a target for the quest
  • You can specify the search distance from the quest giver, as either a max distance or a range
