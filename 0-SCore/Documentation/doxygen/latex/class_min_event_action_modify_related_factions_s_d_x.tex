\doxysubsection{Detailed Description}
Changes the target\textquotesingle{}s relationship with the specified faction, and all the specified related factions (if any), according to the relationships between the faction and the related factions. 

Supported tag attributes\+: 
\begin{DoxyItemize}
\item target  The entity target(s), to change relationships with the factions. Valid target values\+: \char`\"{}self\char`\"{}, \char`\"{}other\char`\"{}, \char`\"{}self\+AOE\char`\"{}, \char`\"{}other\+AOE\char`\"{}, \char`\"{}position\+AOE\char`\"{}. In most cases you should use \char`\"{}self\char`\"{} (the player). Required.   
\item range  Maximum range (distance in meters) to include entities as targets. Used only when the target is one of the AOE (Area Of Effect) values. Optional.   
\item target\+\_\+tags  Adds entities with these tags, to the list of entity targets. Special tag values\+: \char`\"{}party\char`\"{}, \char`\"{}ally\char`\"{}, \char`\"{}enemy\char`\"{}. Optional.   
\item faction  The primary faction whose relationship will change with the target. Required.   
\item value  The amount to add to the relationship between the target and \char`\"{}primary\char`\"{} faction. Positive numbers represent amity, negative numbers represent antagonism. It takes 200 points to change a faction relationship entirely (e.\+g. from \char`\"{}neutral\char`\"{} to \char`\"{}like\char`\"{}). Required.   
\item related  One or more related factions, whose relationships will change with the target according to each one\textquotesingle{}s relationship with the primary faction. It should be a comma-\/separated list of faction names (without spaces). You can not include either the target\textquotesingle{}s faction, nor the \char`\"{}primary\char`\"{} faction. Optional; if not specified, no faction relationships are affected, other than the one between the target and primary faction. In this case, it behaves exactly like the Modify\+Faction\+SDX action.   
\item scale  A ratio by which to scale the target relationship to related factions. Suggested values are between 0 and 1 (exclusive). Optional; defaults to 1. (Ignored if \char`\"{}related\char`\"{} is not specified.)   
\end{DoxyItemize}
