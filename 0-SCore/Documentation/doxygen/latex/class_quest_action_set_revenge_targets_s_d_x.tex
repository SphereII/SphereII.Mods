\doxysubsection{Detailed Description}
This quest objective sets the revenge targets of all entities in range. 

The revenge targets can be set to either the owner of the quest, or to a random member of the quest owner\textquotesingle{}s party which shared the quest. Party members must be in range of the quest. This is set in the \char`\"{}id\char`\"{} attribute. If omitted, defaults to the quest owner only. 

The value attribute is the distance, and can be either a number or the string \char`\"{}location\char`\"{}. If a number, the distance is the diameter (not radius!) around the player, in blocks. If the string \char`\"{}location\char`\"{}, the distance is size of the quest location (usually the POI). If omitted, or if the value is \char`\"{}location\char`\"{} and it can\textquotesingle{}t get the quest location, it defaults to a value of 15 blocks. 

Revenge targets will be set on all living entities in range, except for player entities, entities that the player has hired, or the hires of other players that are allies. Only entities that are awake will have their revenge targets set. 

This sets the revenge targets of all entities within a 20 block area, to the quest owner, and does so in phase 4 of the quest.
